# Task 2- Fundamentals of Operating Fundamentals
  # Overview

This repository covers core Operating System security concepts using Linux and Windows. It focuses on safe system configuration, access control, and OS hardening fundamentals essential for cybersecurity and penetration testing.

# Environment

Ubuntu Linux (Virtual Machine)

Windows Security & Windows Defender

Virtual Machine Workstation

# Key Topics

Virtual machines and environment isolation

User accounts, permissions, and access control

Linux file permissions (chmod, chown)

Administrator vs standard user privileges

Firewall configuration (UFW / Windows Firewall)

Process and service monitoring

Disabling unnecessary services

OS hardening best practices

# Key Takeaways

Virtual machines enable safe security testing

Least privilege reduces system risk

Firewalls and service hardening minimize attack surface

OS hardening strengthens overall system defense
